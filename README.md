# 🧰 xuwei_tools · 许惟的 AI 工具合集

> 一些我自己在用、能让 AI 真正干活的小工具 / Skill。会陆续往里加。
>
> A growing collection of practical AI tools & skills I actually use — will keep adding.

这些工具大多是 **Claude Code / Cursor 等可直接挂载的 Skill / 规则**:把某件事的"做法和审美"固定成规则,让 AI 按规则稳定产出,而不是每次重新教它。

## 工具清单 · Tools

| 工具 | 说明 | What |
|------|------|------|
| [**diagram-generator**](diagram-generator/) | 一句话让 AI 画出排版好、配色统一的架构图 / 流程图 / 关系图 / 对比图(SVG → 高清 PNG) | One sentence → clean, well-aligned diagrams |
| _更多陆续加…_ | | _more coming…_ |

## 怎么用 · How

每个工具是一个独立目录,点进去看各自的 README。多数工具把 `SKILL.md` 挂到你的 AI 编程工具里即可(或直接把内容贴给 AI 当系统提示)。
